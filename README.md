# Laravel Sales Dashboard

Dashboard Penjualan berbasis Laravel untuk memantau performa penjualan, menampilkan grafik tren, ringkasan, dan tabel transaksi.

---

## Fitur

- Filter penjualan berdasarkan tanggal
- Ringkasan total penjualan, total transaksi, produk terjual, rata-rata per transaksi
- Grafik tren penjualan dan penjualan per produk
- Responsif dan mendukung dark mode
- Dibangun menggunakan Laravel, Bootstrap, Chart.js, Flatpickr, dan Font Awesome

---
